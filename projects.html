One Month Project - ACM Website

Fall 2019 - UAH ACM

What is it?

For the Fall 2019 semester, we will be having a one month project where ACM members will be contributing to the UAH ACM website. Members will be given only one month in the beginning of the semester to flesh out personal membership pages.

Rules

•Webpages cannot go over a reasonable amount of storage (including site assets, such as pictures or videos)

•No Bitcoin miners or other software that will adversely affect the viewer of the webpage

•Content may not break the ACM Code of Ethics or the ACM Harassment policy in any way.

•Sites must include a clear link back to the UAH ACM Home page uah.acm.org

•UAH ACM Leadership reserves the right to request changes be made for breaking any of these rules or any other situations that may not be addressed by these rules. We also reserve the right to change these rules at any time.

•Workshops will be held with topics including: Introduction to HTML + CSS, Introduction to Github, and Introduction to JavaScript

Last Years Projects:

CS Club Bot

The Discord Bot project is an ongoing project that the ACM hosts every semester. The Discord Bot lies in our ACM Discord server, and handles miscellaneous tasks, such as XP for being active, a reputation system for users who contribute to the shared knowledge pool, seasonal events, and more. It is written primarily in C#, with other aspects, such as a SQL database. Each semester, the Discord Bot undergoes more revisions. Therefore, we are constantly seeing new features!

BattleCode 2019

BattleCode is a competition where teams from different schools write AIs that compete against each other in custom games. In 2018, we sent our first team to compete in BattleCode to write the best algorithm to defeat the other teams. This competition helped with learning all aspects of the coding process, including planning, collaboration, implementation, testing, and debugging. The final program contained several key features including basic AI to determine strategy, pathfinding to best navigate the field, and a communications protocol to convey information between different parts of the program.

Leadership Lead Projects:

Club Registration App

This app is a simple GUI that is used at ACM events. The application records people's name and email address and compiles this data into a local file on the computer which is then transformed into a mailing list. Written entirely in C#, it uses WinForms for the GUI and features very simple code. Project created by Phil Lane..

Arduino Night Light

Learn how to use an Arduino Uno and LEDs to make a rainbow desk lamp. Project uses Tinkercad to 3D model a case for the Arduino and a Makerbot 3D printer to print the case. Project created by Liv Weaver.

PoseNet

PoseNet was a Convolutional Neural Network for the Air Force Research Lab in Dayton, Ohio. It takes in images of landmarks and outputs the camera's distance from the landmark and the orientation of the camera. Project created by Trystan May.

ABOUT UAH ACM

UAH ACM is a student chapter formed in 2018 that aims to create a positive environment for out of classroom learning about computing. ACM gives members the opportunity to work on passion projects in a team setting. We provide a space and community to hone your craft, be creative, and create life long friendships!
